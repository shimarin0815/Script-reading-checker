# 台本読み上げ時間チェッカー (Script Reading Time Checker)

## 概要

これは、配信者やナレーター、プレゼンターの方向けのシンプルなツールです。
あなたが用意した**台本のテキストを貼り付けるだけ**で、想定される読み上げ時間を\*\*「標準」「早口」「ゆっくり」\*\*の3パターンで即座に計算します。

このアプリは、**単一のHTMLファイルのみ**で動作する「シングルページアプリケーション（SPA）」として作られています。

## プレビュー（アプリアイコン）

（ダークモードを基調としたモダンなデザインです。上部に「ゆっくり」「標準」「早口」の3つの時間（00分00秒）が表示され、その下に台本を貼り付けるための大きなテキストエリアが配置されています。）

## 主な機能

  * **瞬時の計算:** テキストエリアに文字を入力（または貼り付け）すると、リアルタイムで時間が計算・表示されます。
  * **3パターンのWPM:** 以下の3つの速度（WPM = 分速文字数）で時間を算出します。
      * **ゆっくり:** 250文字/分 （聞き取りやすさ重視、朗読など）
      * **標準:** 300文字/分 （一般的なアナウンサー、プレゼン）
      * **早口:** 400文字/分 （情報量の多い動画、速報など）
  * **インストール不要:** 1つのHTMLファイルだけで完結しているため、ダウンロードしてブラウザで開くだけで使えます。

## 使い方（起動方法）

このツールを使うのに、特別な準備やインストールは**一切不要**です。

1.  `checker.html`（または、あなたが保存した`.html`ファイル）を見つけます。
2.  そのファイルを**ダブルクリック**します。
3.  自動的にあなたのPCのWebブラウザ（Chrome, Edge, Safariなど）が起動し、アプリが表示されます。
4.  あとは、表示されたテキストエリアに台本を貼り付けるだけです。

## (おまけ) WPM（分速文字数）の調整方法

もし、あなた自身の読み上げ速度に合わせて計算の基準（WPM）を変更したい場合は、HTMLファイルを直接編集することで簡単にカスタマイズできます。

1.  `checker.html` ファイルを、VS Codeや「メモ帳」などのテキストエディタで開きます。

2.  ファイルの下の方（`<script>`タグの中）にある、以下の部分を探してください。

    ```javascript
    // 「分速文字数（WPM）」をあらかじめ決めておく
    // この数字を変えれば、計算の基準を変えられます
    const WPM_SLOW = 250;   // ゆっくり
    const WPM_NORMAL = 300; // 標準
    const WPM_FAST = 400;   // 早口
    ```

3.  `WPM_SLOW`, `WPM_NORMAL`, `WPM_FAST` の右側にある\*\*数字（250, 300, 400）\*\*を、お好みの数値に変更します。

4.  ファイルを上書き保存します。

5.  再び `checker.html` をブラウザで開き直すと、変更した数値で時間が計算されるようになります。
